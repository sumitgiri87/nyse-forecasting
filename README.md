# NYSE — Next-Day / Next-Week Stock Prediction

**Project goal:** Build a simple, reproducible baseline that predicts (a) whether a stock's closing price will go up/down next day (classification) and (b) the next-week return (regression). Keep the project open for other experiments via PRs.

---

## 1. Problem statement 

**Primary problem:** Given historical daily price data (open, high, low, close, volume) and optional company fundamentals, predict:

1. **Next-day direction (classification)** — will the closing price increase (1) or decrease (0) tomorrow?
2. **Next-week return (regression)** — the percentage return over the next 5 trading days.

These predictions support short-horizon trading signals, ensemble strategies, or risk-flagging. We start with a baseline that is easy to reproduce and interpret.
